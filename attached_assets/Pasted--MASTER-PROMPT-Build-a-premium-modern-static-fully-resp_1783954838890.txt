# MASTER PROMPT

Build a premium, modern, static, fully responsive website for the startup competition **"Biznovator - Innovation Seeds Tomorrow"** hosted as part of **SPROUT IT 2025** by **Symbiosis Centre for Information Technology (SCIT), Pune**.

The website should look like it belongs to a global startup summit (similar quality to Apple, Stripe, Linear, Vercel, Notion, Framer, and Web Summit websites).

---

## Tech Stack

Use only:

* HTML5
* CSS3
* Vanilla JavaScript

No backend.

No React.

No frameworks.

No Bootstrap.

No Tailwind.

Everything should work by simply opening index.html.

Organize files as:

/index.html
/style.css
/script.js
/assets/
images/
icons/

---

## Design Style

Theme:

Future Technology + Startup + Innovation + AI + Venture Capital

The UI should feel like:

* Apple
* Stripe
* Linear
* Vercel
* Framer
* OpenAI
* GitHub Universe

Avoid bright colors.

Use elegant dark gradients.

Use glassmorphism.

Use soft glow effects.

Use smooth shadows.

Rounded corners (16–24px).

Modern typography.

Minimalistic.

Premium.
